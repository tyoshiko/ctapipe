DataVolumeReductor¶

class ctapipe.image.DataVolumeReductor(config, tool, **kwargs)[source]¶

Bases: ctapipe.core.component.Component

Base component for data volume reductors.

Parameters:

config : traitlets.loader.Config

Configuration specified by config file or cmdline arguments. Used to set traitlet values. Set to None if no configuration to pass.

tool : ctapipe.core.Tool

Tool executable that is calling this component. Passes the correct logger to the component. Set to None if no Tool to pass.

kwargs

Attributes

extracted_samples (ndarray) Numpy array containing True where the samples lay within the integration window, and False where the samples lay outside. Has shape of (n_chan, n_pix, n_samples).
peakpos (ndarray) Numpy array of the peak position for each pixel. Has shape of (n_chan, n_pix).
neighbours (list) List of neighbours for each pixel. Changes per telescope.
